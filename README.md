Bank Client Management System
console-based C++ application to manage bank clients.  

Features:
- Add new client.
- Delete client.
- Update client information.
- View all clients.
- Search for clients by account number.
- Deposit and withdraw transactions.
- Display total balances.
- 
Data Storage:
Client data is stored in `Clients.txt` using a simple text format. Each record is separated by a custom delimiter.
